### Starting the App

When you first open the application, you'll see the **Home Page** with:
- Your WheelsDoc AutoCare logo
- Feature highlights
- "Sign In" and "Get Started" buttons

### For New Users

1. Click **"Get Started"**
2. Fill out the signup form
3. After signup, you'll be taken to login
4. Sign in with your credentials
5. You'll automatically land on your dashboard

### For Existing Users

1. Click **"Sign In"**
2. Enter your email and password
3. You'll automatically land on your dashboard

### While Using Dashboards

**Every dashboard has:**
- **Home Button** (in the header) - Click to return to the home page
- **User Menu** (top right) - Shows your name, email, and role
- **Logout Button** (in user menu) - Click to sign out

### Navigation Buttons

| Where | Button | What It Does |
|-------|--------|--------------|
| Home Page (logged out) | "Sign In" | Opens login page |
| Home Page (logged out) | "Get Started" | Opens signup page |
| Home Page (logged in) | "Go to Dashboard" | Opens your role-specific dashboard |
| Home Page (logged in) | "Logout" | Signs you out |
| Login Page | "Back to Home" | Returns to home page |
| Signup Page | "Back to Home" | Returns to home page |
| Any Dashboard | "Home" (header) | Returns to home page |
| Any Dashboard | "Logout" (user menu) | Signs you out and returns to home |
